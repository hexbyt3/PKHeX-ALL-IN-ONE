You could do a custom scale #, a range, or KISS add a field in ``Search Criteria`` for ``Size Marking`` with the options ``None``,``Any``,``Mini``,``Jumbo`` -

``none`` - if scale != 0 or 255
``any`` - leave the code as is, literally does nothing 
``Mini`` if scale = 0
``Jumbo`` if scale = 255
